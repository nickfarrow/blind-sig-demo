What are blind signatures?
With regular schnorr signatures, some message is used to create a challenge which is then signed using a private key and a nonce.

Blind signatures allow for a user to disguise a message and then request a server to sign this disguised message. The user can then take this signature and unblind it, resulting in an signature that is valid for the original message. This is useful for protecting user privacy, where a server can act as an authority while remaining blind to the exact activity occuring.

Hint: Expand \(s'\) from definitions and chug away

Starting with the unblinded signature: $$\begin{aligned} s' &= s + \alpha + c * t \\ &= (k + c'*x) + \alpha + c * t \\ &= k + (c+\beta)*x + \alpha + c * t \\ &= (k + \alpha + \beta*x) + c*(x + t) \end{aligned}$$ Hence $$\begin{aligned} s'*G &= ((k + \alpha + \beta*x) + c*(x + t))*G \\ &= (k + \alpha + \beta*x)*G + c*(x + t)*G \\ &= k*G + \alpha*G + \beta*(x*G) + c*((x*G) + (t*G)) \\ &= (R + \alpha*G + \beta*X) + c*(X + t*G) \\ &= R' + c*X' \end{aligned}$$
